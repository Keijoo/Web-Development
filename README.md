# Prizm - Web Development Coursework

A multi-page website project ("Prizm: In Pursuit of Freeing One's Mind") built for university coursework, featuring Home, About, Gallery, and Contact pages.

## About

This was built as part of my university's web development module. Note: as this is coursework, I've kept sharing of the full assignment brief and content to a minimum in line with academic policy - this repo mainly serves as a record of the front-end build.

## Features

- Multi-page site: Home, About, Gallery, Contact
- Custom navigation and layout
- Image gallery and embedded video

## Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript

## Running Locally

```bash
git clone https://github.com/Keijoo/Web-Development.git
cd Web-Development/website
# open Index.html in your browser
```
